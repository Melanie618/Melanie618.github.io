## Use

### 日常发布

发布时，推送至 GitHub 的 hexo 分支备份，同时生成静态页面推送至 GitHub 的 master 分支

```sh
hexo clean & hexo generate & hexo deploy
# or
# hexo clean & hexo g & hexo d
```
