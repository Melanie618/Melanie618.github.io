# css 长度单位
**绝对长度单位：**
绝对长度单位表示为一个固定的值，不会改变 **不利于页面渲染**
- in 英寸
- cm 里面
- mm 毫米
- pt
- pc

**相对长度单位：**
长度单位会随着他的参考值变化
- px 像素
- em 元素的字体高度
- % 百分比
- rem 根元素的font-size
- vw 视窗宽度，1vw == 视窗宽度的 1%
- vh 视窗高度，1vh == 视窗高度的 1%
